# CAC Registrations Port Harcourt

A premium, responsive website for **CAC Registrations Port Harcourt**, a Nigerian business led by Mr. David providing CAC Business Name, Company, Club, Association, and Union Registration services.

Built with **React + TypeScript + Vite + Tailwind CSS**.

## Features

- Light / Dark / System theme support with persistence
- Sticky navbar with blur-on-scroll + mobile menu
- Floating WhatsApp button with pulse animation and tooltip
- Sections: Hero, Services, Why Choose Us, How It Works, About, Reviews, CTA, Contact
- Per-service WhatsApp buttons with pre-filled messages
- Contact form backed by Supabase (with copy-address button)
- SEO: meta tags, Open Graph, Twitter cards, LocalBusiness structured data (JSON-LD)
- Scroll-reveal animations with reduced-motion support
- Fully responsive across all breakpoints
- Blue, white, and green color palette

## Tech Stack

- React 18 + TypeScript
- Vite 5 (with manual chunk splitting for performance)
- Tailwind CSS 3
- Lucide React icons
- Supabase (contact form storage)

## Getting Started

```bash
npm install
npm run dev
```

Set your Supabase credentials in `.env` (see `.env.example`).

## Build

```bash
npm run build
npm run preview
```

## Deployment

Connected to Netlify via GitHub — every push to `main` triggers an automatic deployment.
