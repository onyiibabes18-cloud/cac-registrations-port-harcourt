# CAC Registrations Port Harcourt

A premium, responsive website for **CAC Registrations Port Harcourt**, a Nigerian business led by Mr. David providing CAC Business Name, Company, Club, Association, and Union Registration services.

Built with **React + TypeScript + Vite + Tailwind CSS**.

## Features

- Light / Dark / System theme support with persistence
- Sticky navbar with blur-on-scroll + mobile menu
- Floating WhatsApp button with pulse animation
- Sections: Hero, About, Services, Why Choose Us, How It Works, Google Reviews, FAQ, Contact
- Contact form backed by Supabase
- SEO: meta tags, Open Graph, Twitter cards, LocalBusiness structured data
- Scroll-reveal animations with reduced-motion support
- Fully responsive across all breakpoints
- Blue, white, and green color palette

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Lucide React icons
- Supabase (contact form storage)

## Getting Started

```bash
npm install
npm run dev
```

Set your Supabase credentials in `.env` (see `.env.example`).

## Build

```bash
npm run build
npm run preview
```
